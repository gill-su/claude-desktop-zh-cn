# Claude Desktop 繁體中文安裝（同事看這份）

把 Windows 版 Claude Desktop 的介面換成**繁體中文（台灣）**。

> 適用：Microsoft Store 版的 Claude Desktop（一般從官網/商店安裝的就是這個）。

---

## 一、安裝步驟（最快）

1. 按右上角綠色 **`Code` → `Download ZIP`** 下載整包，解壓縮到任意資料夾。
   （或用 `git clone https://github.com/gill-su/claude-desktop-zh-cn.git`）
2. **完全關閉 Claude Desktop**：右下角系統匣圖示 → 右鍵 → Quit。
3. 雙擊 **`一鍵安裝-繁體中文.bat`**。
4. 跳出 **UAC「使用者帳戶控制」** 時按 **「是」**。
5. 等它跑完（最後一步要重算 `Claude.exe` 約 220MB，**可能要數分鐘**，請耐心等，不要手動去開 Claude）。
6. 完成後 Claude 會自動開成中文。若仍是英文：左下角帳號 → **Language** → 選 **繁體中文（台灣）**。

---

## 二、注意事項

- 本腳本用的是**官方登入模式**（會修改 `app.asar`，語言選單才會出現繁體中文）。
  代價：**Cowork 沙箱／工作區會無法使用**，且 `Claude.exe` 數位簽章會被改寫。
- **Claude 更新後補丁會被洗掉**，更新完再跑一次 `一鍵安裝-繁體中文.bat` 即可。
- 腳本做了什麼：自動提權 → 徹底關閉 Claude（避免檔案被鎖而卡住）→ 安裝 zh-TW →
  設定語言 → 重新開啟 Claude。原始檔會自動備份在
  `...\WindowsApps\Claude_*\app\resources\.zh-cn-backups\`。

---

## 三、想還原成英文 / 移除補丁

用系統管理員開 PowerShell，在本資料夾執行：

```powershell
powershell -ExecutionPolicy Bypass -File scripts\install_windows.ps1 uninstall
```

或直接跑 `install-windows.bat` 後選 **模式 3（恢復原樣 / 卸載補丁）**。

---

## 四、跑不動 / 卡住怎麼辦

- **卡在 `[7/8]` 很久沒動**：通常是 Claude 又被開起來鎖住檔案。請把所有 Claude 視窗關掉、
  確認系統匣沒有 Claude 圖示，然後重跑一次 `一鍵安裝-繁體中文.bat`。
- **雙擊 .bat 一閃就關**：改用右鍵以系統管理員開 PowerShell，手動執行：
  `powershell -ExecutionPolicy Bypass -File 一鍵安裝-繁體中文.ps1`
- **公司電腦擋下載/執行**：請找 IT 允許執行 PowerShell 腳本，或改用手動方式
  （見原專案 `README.md` 的 Windows 安裝段落，選模式 2 + 語言 2）。
